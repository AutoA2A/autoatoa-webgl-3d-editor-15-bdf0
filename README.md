# WebGL 3D 模型编辑器 项目说明

## 网站简介
本项目是一个基于 WebGL 技术的在线三维模型编辑器，旨在提供轻量、直观的模型查看与基础编辑功能。用户无需安装专业软件，即可在浏览器中直接操作 3D 模型（如旋转、缩放、平移，以及简单的材质与灯光调整）。项目由多个 AI 模型协同开发，共同完成了从架构设计、前端交互、渲染逻辑到部署测试的全流程，是一个典型的多智能体协作实践案例。

## 页面与功能
网站共包含 5 个核心页面：
1.  **index.html（主页与模型视口）**：核心操作界面，包含 3D 渲染画布、基本控制面板，以及模型导入/导出功能入口。
2.  **model-library.html（模型库）**：展示预设或用户上传的模型列表，提供快速预览和选择功能。
3.  **material-editor.html（材质编辑）**：提供材质属性（如颜色、粗糙度、金属度）的调节界面，实现实时预览。
4.  **lighting-setup.html（灯光设置）**：允许用户调整场景灯光类型、强度、颜色和方向。
5.  **help.html（帮助与文档）**：提供操作指南、功能说明与常见问题解答。

## 多 AI 协作与验收
本项目采用多 AI 协作开发模式，模拟了团队协作流程。不同 AI 分别负责：**系统架构设计**、**WebGL 渲染核心编写**、**前端 UI 组件实现**、**测试与部署脚本生成**。各模块通过明确定义的接口进行集成，并由另一个独立的 AI 作为“验收者”，对代码规范、功能完整性和整体体验进行审查与反馈。这一流程验证了多智能体在复杂软件工程中的协同能力。

## GitHub Pages 部署与访问（入口 index.html）
项目已通过 GitHub Pages 部署为静态网站，所有页面及资源均可通过网络直接访问。主入口文件为 `index.html`，用户从此页面进入编辑器主界面，并可通过导航栏跳转至其他功能页面。在线体验链接可通过项目的 GitHub 仓库获得。
